# KilimoSmart Design Tokens

## Typography
- **Family**: Inter
- **Weights**: 400 (Regular), 500 (Medium), 600 (SemiBold), 700 (Bold)

## Spacing Scale
- Base unit: 8px
- `space-1`: 8px
- `space-2`: 16px
- `space-3`: 24px
- `space-4`: 32px
- `space-5`: 40px
- `space-6`: 48px
- `space-8`: 64px

## Shape System
- **Buttons**: 4px (rounded-sm)
- **Cards**: 16px (rounded-2xl)
- **Pills/Filters**: 9999px (rounded-full)

## Shadows
- **Soft Lift (Cards)**: `0 4px 6px -1px rgba(0, 0, 0, 0.1), 0 2px 4px -1px rgba(0, 0, 0, 0.06)`

## Colors
- **Primary**: Harvest Green `#1E6B45`
- **Status - Pending**: Amber Ochre `#B9762A`
- **Status - Rejected**: Bloom Deep `#B3262B`
- **Status - Accepted**: Harvest Green `#1E6B45`
- **Map/Weather**: Storm Teal `#1F4E5B` (reserved only for weather/map content)
